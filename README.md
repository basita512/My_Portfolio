## Setup

```bash
npm install
cp .env.example .env.local   # add your Web3Forms key & resume link
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

| Variable | Purpose |
|---|---|
| `NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY` | Contact form API key |
| `NEXT_PUBLIC_RESUME_LINK` | Resume download URL |

## Deploy

Push to GitHub → import on [Vercel](https://vercel.com) → add env variables → deploy.
